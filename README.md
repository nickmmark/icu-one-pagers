## icu-one-pagers for COVID-19
One page summary of critical care topics relevant to care of patients in the intensive care unit. This repository was created for facilitate access and allow crowdsources updates relevant to COVID-19.

For the original document as well as for other #ICUonepagers by @nickmmark please see [www.iceonepager.com](www.icuonepager.com).

### translations
* this is a crowd sourced effort; if you would like to contribute please download the latest PPT from [www.iceonepager.com](www.icuonepager.com). Translate and validate. Upload the .pdf and .png files to this github using the file naming conventions below. Do NOT change the version number (there is one common version across all translations).
